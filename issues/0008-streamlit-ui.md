# Streamlit UI (optional)

**Summary**
Add a Streamlit-based frontend that reads `data/students.json` and provides a polished search UI with student cards, activity history, and donate modal.

**Acceptance criteria**
- `app.py` or `streamlit_app.py` renders search input and results
- Reusable UI components under `src/ui/` (layout, cards, banner)
- Styling in `src/ui/styles.css` and responsive design

**Labels:** `frontend`, `feature`, `ui`
